SKILLS

  • Design

    デザインの基礎を習得しております。 目的やご希望に合わせたデザインを考え、ワイヤーフレーム・デザインカンプを一から作成することが可能です。

  • Photoshop

    photoshopを駆使して写真の色補正や、複数枚の画像を組み合わせた合成加工まで行います。また人物や小物などの切り抜き作業が可能です。

  • illustrator

    illustratorでバナー広告制作を行います。また、文字を加工しデフォルトに無い文字や、装飾を作成する作業が可能です。

  • HTML / CSS

    Webサイトを一から構築するための、HTML・CSSコーディングが可能です。メインで使用しているエディタはVisual Studio Codeです。

  • スライダー・ハンバーガーメニュー・モーダルウィンドウなど、webサイトに簡単な動きをつけるための、jQueryカスタマイズが可能です。

  • レスポンシブ

    パソコン、タブレット、携帯など各デバイスの画面サイズに応じて、デザインの作成をすることや、コーディングが可能です。
